# P-PROGRAMS (ATA 06–12)

**Axis:** P — Programs  
**ATA Scope:** Chapters 06–12  
**Last Updated:** 2026-01-27

---

## Overview

The P-PROGRAMS axis covers ground handling and servicing procedures, including dimensions, lifting, leveling, towing, parking, placards, and general servicing operations.

## Key Relationships

- **ATA 06** dimensions are referenced by all structural chapters
- **ATA 12** servicing procedures are critical for H₂ fueling operations
- These chapters interface heavily with **I-INFRASTRUCTURES** for ground support equipment
