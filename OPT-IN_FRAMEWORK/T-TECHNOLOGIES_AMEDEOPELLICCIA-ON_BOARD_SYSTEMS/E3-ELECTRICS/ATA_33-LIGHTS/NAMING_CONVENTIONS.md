## ID Grammar Rules

### General Format

```
{PREFIX}-{CHAPTER}-{SECTION}-{SUBJECT}-{TYPE}-{SEQUENCE}
```

### Examples

- `REQ-33-00-00-SRS-001` — System Requirement Specification
- `DES-33-10-00-DDD-001` — Design Description Document
- `TEST-33-00-00-TP-001` — Test Procedure
