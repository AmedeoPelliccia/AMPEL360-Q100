# E3-ELECTRICS

**Electrical Systems (ATA 33, 39)**

This axis of the OPT-IN Framework covers **electrical systems and components** for the AMPEL360 Q100 aircraft — lighting systems and electrical/electronic panels.

> **Governance Note:** ATA chapters 33 and 39 are standard ATA chapters for Electrical Systems. They represent *"what provides electrical function"* — lights, panels, and multipurpose electrical components.

---

## ATA Chapters in This Axis

| Chapter | System | Description |
|---------|--------|-------------|
| ATA 33 | Lights | Interior and exterior lighting systems |
| ATA 39 | Electrical/Electronic Panels | Multipurpose electrical panels and components |

---

## Operational Rule

> **E3-ELECTRICS (T-TECHNOLOGIES / electrics)**: *"lo que proporciona funciones eléctricas"* → lights, panels, electrical components.

If the artifact is **electrical systems/components** → place in **E3-ELECTRICS**.
