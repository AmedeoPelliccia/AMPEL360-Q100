# I2-INTELLIGENCE Subdomain

**Axis:** T-TECHNOLOGIES  
**Subdomain Code:** I2  
**ATA Chapters:** 95, 97

---

## Overview

The Intelligence subdomain covers AI/ML systems and synthetic data validation — novel extensions for the Q100 program addressing learning assurance and data-driven certification.

---

## Q100-Specific Focus

- **Learning Assurance (ATA 95)** — DO-178C/DO-200B extensions for ML
- **Model Validation** — Verification of AI/ML models
- **Synthetic Data (ATA 97)** — Validation of synthetically generated training data
- **Certification Evidence** — ML-specific compliance documentation
