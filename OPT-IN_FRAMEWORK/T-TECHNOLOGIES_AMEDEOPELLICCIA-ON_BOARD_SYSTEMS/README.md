# T-TECHNOLOGIES (On-Board Systems)

**Axis:** T — Technologies  
**ATA Scope:** Chapters 20–80 (On-Board Aircraft Systems)  
**Last Updated:** 2026-01-27

## Overview

The T-TECHNOLOGIES axis is the largest and most comprehensive axis, covering all on-board aircraft systems organized into 15 subdomains based on functional grouping.
