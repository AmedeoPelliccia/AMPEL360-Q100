# C2-CIRCULAR_CRYOGENIC_CELLS Subdomain

**Axis:** T-TECHNOLOGIES  
**Subdomain Code:** C2  
**ATA Chapters:** 28

---

## Overview

The Circular Cryogenic Cells subdomain covers fuel systems with a focus on liquid hydrogen (LH₂) cryogenic storage and distribution for the AMPEL360 Q100 hydrogen-electric aircraft.

---

## Q100-Specific Focus

- **LH₂ Cryogenic Storage** — Vacuum-insulated tanks for liquid hydrogen
- **Fuel Distribution** — Cryogenic fuel lines and components
- **Fuel Quantity Measurement** — Specialized sensors for LH₂
- **Safety Systems** — H₂ leak detection and venting
- **Thermal Management** — Boil-off management and recirculation
