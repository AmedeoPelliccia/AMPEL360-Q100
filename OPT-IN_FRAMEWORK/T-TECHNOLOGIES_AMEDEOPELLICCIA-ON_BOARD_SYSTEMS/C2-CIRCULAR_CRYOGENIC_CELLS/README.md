# C2-CIRCULAR_CRYOGENIC_CELLS

**Fuel Systems (ATA 28)**

This axis of the OPT-IN Framework covers **fuel systems** for the AMPEL360 Q100 aircraft — fuel storage, distribution, dump, and indicating systems. For hydrogen-hybrid aircraft, this includes cryogenic fuel cell integration.

> **Governance Note:** ATA chapter 28 is the standard ATA chapter for Fuel. It represents *"what almacena y distribuye combustible"* — fuel storage, distribution, dump, indicating. For H₂ applications, includes cryogenic cells.

## ATA Chapters in This Axis

| Chapter | System | Description |
|---------|--------|-------------|

## Operational Rule

> **C2-CIRCULAR_CRYOGENIC_CELLS (T-TECHNOLOGIES / fuel)**: *"lo que almacena y distribuye combustible"* → fuel tanks, distribution, cryogenic cells.
