## Content

### 1. General Description

#### 1.1 System Purpose

The Liquid Hydrogen (LH₂) Fuel Storage System (ATA 28-10) provides safe and efficient storage of liquid hydrogen fuel for the AMPEL360 Q100 hydrogen-electric aircraft propulsion system.

#### 1.2 System Features

The fuel storage system incorporates the following key features:

| Feature | Description |
|---------|-------------|
| **Cryogenic Tanks** | Type IV COPV with carbon fiber overwrap |
| **MLI Insulation** | Multi-layer insulation for thermal management |
| **Boil-off Management** | Passive/active system for minimal fuel loss |
| **BWB Integration** | Conformal design integrated with airframe |

### 2. System Architecture

#### 2.1 Tank Configuration

The Q100 fuel storage system consists of:

| Tank | Location | Capacity | Purpose |
|------|----------|----------|---------|
| **Tank 1** | Forward Center | 400 kg LH₂ | Primary storage |
| **Tank 2** | Aft Center | 400 kg LH₂ | Primary storage |

**Total System Capacity:** 800 kg LH₂

#### 2.2 Major Components

1. **Pressure Vessel Assembly**
   - Inner liner (aluminum alloy)
   - Composite overwrap (carbon fiber/epoxy)
   - Boss fittings (stainless steel)

2. **Thermal Insulation System**
   - Multi-layer insulation (MLI)
   - Vapor-cooled shields (optional)
   - Vacuum jacket (ground operations)

3. **Fill/Drain System**
   - Fill port interface
   - Drain port interface
   - Quick-disconnect couplings

4. **Pressure Management**
   - Pressure sensors
   - Relief valves
   - Vent system

5. **Instrumentation**
   - Level sensors
   - Temperature sensors
   - Pressure sensors

### 3. Operating Principles

#### 3.1 Normal Operation

During normal operation, the fuel storage system:

1. Maintains LH₂ at 2-5 bar and 20-30 K
2. Delivers fuel to distribution system on demand
3. Manages boil-off through controlled venting or re-liquefaction
4. Provides quantity and status information to flight deck

#### 3.2 Refueling Operation

Refueling sequence:

1. Connect ground service equipment (GSE)
2. Chilldown transfer lines
3. Transfer LH₂ to tanks
4. Verify quantity and pressure
5. Disconnect GSE

**Nominal Refuel Time:** ≤ 30 minutes (0-100%)
