## ID Grammar Rules

### General Format

```
{PREFIX}-{CHAPTER}-{SECTION}-{SUBJECT}-{TYPE}-{SEQUENCE}
```

### Examples

- `REQ-76-00-00-SRS-001` — System Requirement Specification
- `DES-76-10-00-DDD-001` — Design Description Document
- `TEST-76-00-00-TP-001` — Test Procedure
