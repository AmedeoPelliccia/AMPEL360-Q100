# A2-AVIONICS

**Avionics Systems (ATA 22, 42)**

This axis of the OPT-IN Framework covers **avionics systems** for the AMPEL360 Q100 aircraft — auto flight systems and integrated modular avionics.

> **Governance Note:** ATA chapters 22 and 42 are standard ATA chapters for Avionics. They represent *"what flies and computes"* — autopilot, auto-throttle, IMA, and flight computing.

---

## ATA Chapters in This Axis

| Chapter | System | Description |
|---------|--------|-------------|
| ATA 22 | Auto Flight | Autopilot, auto-throttle, speed/attitude correction |
| ATA 42 | Integrated Modular Avionics | IMA platforms and computing |

---

## Operational Rule

> **A2-AVIONICS (T-TECHNOLOGIES / avionics)**: *"what flies and computes"* → autopilot, autothrottle, IMA.

If the artifact is **auto flight/IMA** → place in **A2-AVIONICS**.
