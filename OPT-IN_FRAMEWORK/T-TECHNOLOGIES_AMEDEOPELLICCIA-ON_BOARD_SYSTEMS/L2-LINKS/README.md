# L2-LINKS

**Navigation & Links Systems (ATA 34)**

This axis of the OPT-IN Framework covers **navigation and positioning systems** for the AMPEL360 Q100 aircraft — flight environment data, attitude/direction, landing aids, and flight management computing.

> **Governance Note:** ATA chapter 34 is the standard ATA chapter for Navigation. It represents *"what navigates and positions"* — flight environment, attitude, landing aids, position determining, flight management.
