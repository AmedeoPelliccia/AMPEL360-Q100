# E1-ENVIRONMENT Subdomain

**Axis:** T-TECHNOLOGIES  
**Subdomain Code:** E1  
**ATA Chapters:** 21, 26, 30, 35-38, 47

---

## Overview

The Environment subdomain covers environmental control systems including air conditioning, pressurization, fire protection, ice/rain protection, oxygen, pneumatic, vacuum, water/waste, and nitrogen generation.
