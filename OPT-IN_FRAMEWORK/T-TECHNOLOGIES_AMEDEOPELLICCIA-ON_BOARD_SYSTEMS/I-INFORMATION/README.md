# I-INFORMATION

**Information Systems (ATA 46)**

This axis of the OPT-IN Framework covers **information systems** for the AMPEL360 Q100 aircraft — airplane information systems, flight deck displays, maintenance information, and passenger cabin systems.

> **Governance Note:** ATA chapter 46 is the standard ATA chapter for Information Systems. It represents *"what processes and displays information"* — general information systems, flight deck, maintenance, and passenger information.

---

## ATA Chapters in This Axis

| Chapter | System | Description |
|---------|--------|-------------|
| ATA 46 | Information Systems | Airplane, flight deck, maintenance, and passenger information systems |

## Operational Rule

> **I-INFORMATION (T-TECHNOLOGIES / information)**: *"lo que procesa y muestra información"* → airplane info systems, flight deck displays, maintenance info, passenger info.

If the artifact is **information processing/display** → place in **I-INFORMATION**.
